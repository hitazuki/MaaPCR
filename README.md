</div>

## 使用说明

### 文件下载

- 在右边release页面 根据操作系统和架构下载压缩包（一般是windows_x86_64
- 首次运行需要安装.net和vc++，压缩包中有安装脚本
- 压缩包已内置python环境，无需额外安装

### 快速开始

- Windows平台运行MaaPCR.exe即可
- 首次运行和使用DMM端可能需要在管理员权限下运行

### 支持平台

#### 安卓模拟器

- 使用前将模拟器分辨率调整为1280x720(DPI 240)
- 控制器类型选择安卓端，当前控制器选择游戏对应实例
- 目前仅测试过mumu模拟器（其他应该也行

#### DMM

- 使用前将窗口分辨率调整为1280x720（拉至最大即为1280x720
- 控制器类型选择桌面端，当前控制器选择游戏窗口
- 目前由于识别不了安卓返回按钮信号，导致部分任务完成后无法返回主页，运行多个任务时可能会阻塞
- 鼠标输入方式默认为"SendMessageWithWindowPos"以减少点击失效的情况，有需要可以在设置页面修改为其他方式
